# 시뮬레이션 무게/관성 조정 — 2026-05-16

## 무엇을 했는지
오늘(2026-05-16) Phase 0 W3 주차의 작업인 "시뮬 무게/관성 조정"을 수행했습니다.
`SO-ARM100/Simulation/SO101/so101_new_calib.xml` 파일 내의 모든 `<body/>` 태그 하위 `<inertial/>` 요소에 대해, 현재는 CAD 기본값을 사용하고 있으나 향후 실측값으로 교체할 예정임을 명시하는 XML 주석을 추가했습니다.

예시:
```xml
      <!-- TODO: mass/inertia — CAD 기본값, 실측값으로 교체 예정 -->
      <inertial pos="0.0137179 -5.19711e-05 0.0334843" mass="0.147" fullinertia="0.000114686 0.000136117 0.000130364 -4.59787e-07 4.97151e-06 9.75275e-08"/>
```
이 작업은 `claude-code` 스킬을 통해 `SO-ARM100/Simulation/SO101/so101_new_calib.xml` 파일을 직접 수정하여 이루어졌습니다.

## 어떻게 검증했는지
`claude-code`의 응답을 통해 7개의 `body` 모두에 주석이 성공적으로 추가되었음을 확인했습니다. `base`, `shoulder`, `upper_arm`, `lower_arm`에는 기존 `<inertial>` 태그가 있었고, `wrist`, `gripper`, `moving_jaw_so101_v1`에는 `<inertial>` 태그가 새로 추가되었습니다.

## 다음 단계로 어떻게 이어지는지
무게/관성 파라미터는 시뮬레이션의 물리적 정확도에 중요한 영향을 미칩니다. 현재는 기본값을 사용하지만, 추후 실측값이 확보되면 해당 주석을 따라 값을 업데이트하여 시뮬레이션의 현실성을 높일 예정입니다.
다음 단계는 "동일 명령에 대한 시뮬 vs 실기 관절각 비교 (목표 ±1°)" 작업을 준비하는 것입니다. 이를 위해서는 `so101_new_calib.xml` 파일을 로드하여 MuJoCo 시뮬레이션을 실행하고 관절 데이터를 추출하는 코드를 작성해야 합니다.
